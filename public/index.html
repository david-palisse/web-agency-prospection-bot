Exporter les commerces sans site internet

Saisissez une ville, puis l'application interrogera Google Places API pour constituer un fichier CSV des commerces trouvés sans site web.

  • Le CSV contient le nom, le type, l'adresse, la note, l'e-mail et le téléphone.
  • L'adresse e-mail est généralement absente de Google Places API : la colonne reste vide.
  • Les résultats dépendent de la couverture Google Places et des quotas de votre clé API.
